    • Hello, I’m marixc 👋

    Welcome to my digital playground. I’m a front-end developer, design explorer, and lifelong learner who thrives on clean code and creative challenges. This blog is where I think out loud, build in public, and share what I'm learning.

    👨‍💻 About Me

    I started coding as a teenager out of curiosity. Years later, I’ve built small apps, prototypes, design systems, and countless side projects. I care about accessibility, performance, and expressive UI.

    When I'm not building for the web, I’m probably sketching out UI ideas, reading up on system design, or tweaking VS Code themes.

    🧠 What I Work With

    • HTML, CSS, JavaScript (ES6+)
    • React, Astro, Vue
    • Tailwind CSS, Sass, Styled Components
    • Git, GitHub, Netlify, Vercel
    • Figma, Adobe XD, Affinity Designer

    🌱 My Philosophy

    Build things you want to see exist. Don’t wait until it’s perfect. Share early, iterate often.
